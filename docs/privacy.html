Privacy Policy — Wordle Wizard (Unofficial)

Last updated: 2026-01-28

Summary

  • This extension does not collect or transmit any personally identifiable information to external servers.
  • All data is processed locally in your browser.

Data usage

  • The extension stores a small dictionary and a bundled meanings map in chrome.storage.local to provide quick definitions and suggestions. This data is bundled with the extension and may be persisted in local storage for offline use.
  • No telemetry, analytics, or external network calls are made by default that would send user data off-device. The extension may fetch packaged resources (bundled JSON files) from its own package to initialize local storage.
  • On-demand definition lookups: The extension may make optional, user-initiated requests to external services (for example, Wiktionary) when you ask it to look up a definition. These requests are made only as a result of explicit user actions (clicking a word/lookup button) and are not sent automatically or without consent.

Permissions and what they’re used for

  • scripting: inject the helper UI and scripts into matched pages; content scripts run on the declared host pages via manifest host permissions.
  • clipboardWrite: allow explicit user-initiated copy actions (e.g., "copy suggestion").
  • storage: persist user preferences and local caches (wordlists, UI settings) for faster, offline-friendly operation.
  • alarms: optional local housekeeping tasks like cache refreshes (user-controlled).
